# Image Folder Structure

Simply add your images to the appropriate folders with simple numbering. **No code changes needed!**

## Folder Organization

### 📸 Photography
**Folder:** `images/photography/`
- Add files: `image1.jpg`, `image2.jpg`, `image3.jpg`, `image4.jpg`, `image5.jpg`, `image6.jpg`
- These will display in the Photography gallery tab

### 🎨 Graphics
**Folder:** `images/graphics/`
- Add files: `image1.jpg`, `image2.jpg`, `image3.jpg`, `image4.jpg`, `image5.jpg`, `image6.jpg`
- These will display in the Graphics gallery tab

### 🤖 AI Art
**Folder:** `images/ai-art/`
- Add files: `image1.jpg`, `image2.jpg`, `image3.jpg`, `image4.jpg`, `image5.jpg`, `image6.jpg`
- These will display in the AI Art gallery tab

### 🎯 Extracurricular
**Folder:** `images/extracurricular/`
- Add files: `image1.jpg`, `image2.jpg`, `image3.jpg`, `image4.jpg`
- These will display in the Beyond the Desk / Extracurricular section

## That's it!
Just drop your JPG images into the appropriate folders with the naming scheme above. The website will automatically display them in the correct gallery sections. No code modifications needed!
